# JARVIS OS - Tablet & Mobil Uyumlu Yapay Zeka Asistanı

Bu proje, tabletinizden veya telefonunuzdan tarayıcı aracılığıyla erişebileceğiniz, sesli komut alıp sesli yanıt verebilen Iron Man temalı bir **J.A.R.V.I.S.** web uygulamasıdır.

## Kurulum ve Dağıtım (Deployment)

1. Bu depoyu (repository) GitHub hesabınıza yükleyin.
2. **Render.com** veya benzeri bir bulut sunucu platformuna gidin.
3. GitHub hesabınızı bağlayarak bu projeyi bir **Web Service** olarak seçin.
4. Sunucu ayarlarında Environment Variables (Ortam Değişkenleri) kısmına `OPENAI_API_KEY` değişkenini ve kendi anahtarınızı ekleyin.
5. Dağıtım tamamlandığında sunucunun size verdiği linke tabletinizden girip Ark Reaktörüne dokunarak konuşmaya başlayın!
